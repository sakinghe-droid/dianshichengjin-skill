# 三方交叉验证工作流

当需要对单只个股做高置信度判断时，使用三源交叉验证：

## 工作流

```
① 点石成金 trade_orchestrator  →  E5位置 + 六维 + 经典 + 共识 + 仓位
② mx-xuangu 单日DDX           →  主力净额 / 流通市值 → 验证引擎DDX
③ 扣子 五维分析                →  DDX多周期 + 资金拆解 + 基本面 + 概念纯度
```

## 各源强项

| 源 | 核心能力 | 盲区 |
|------|---------|------|
| 点石成金 | E5位置·仓位天花板·ABCD象限·博弈六维 | DDX默认值·无资金拆解·无基本面 |
| mx-xuangu | 单日真实DDX·流通市值 | 无多周期趋势·无资金拆解 |
| 扣子 | DDX多周期(10/5/3日)·超大单/大单/游资分层·PE/Q1/现金流·概念纯度 | 无E5位置·无ABCD象限 |

## 验证清单

| 检查项 | 源 | 通过条件 |
|--------|-----|---------|
| E5位置 | 点石成金 | ≤65%中位以下 |
| ABCD象限 | 点石成金 | ≠D象限 |
| 六维评分 | 点石成金 | ≥60 |
| DDX趋势 | 扣子/mx-xuangu | 逐级递增≥2周期，任一周期不为负 |
| 资金拆解 | 扣子 | 超大单+大单合计净流入，非游资独舞 |
| 基本面 | 扣子 | PE不极端·Q1不亏损·现金流不转负 |
| 概念纯度 | 扣子 | 主营占比>60%在主线方向 |
